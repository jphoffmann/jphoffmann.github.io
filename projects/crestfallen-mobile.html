Crestfallen Mobile

Crestfallen Mobile is a 2D-Platformer made within the "Multimedia" course at TU Vienna. Its goal was to learn the handling of multimedia content on the Android platform. It got selected for the Top-10 Hall of Fame from 57 games!

Notable learned experiences:

  • Was responsible for implementing all gameplay systems, a custom character controller, the scene structure, data persistence and UI.
  • Using the Android Jetpack Suite for data persistence and responsive UI layout components.
